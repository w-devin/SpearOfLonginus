# Spear Of Longinus

## 名字起源

- Spear of Longinus(隆基努斯之枪), 这里有两个含义, 一个是弑杀耶稣的长枪, 另外就是 EVA 中的命运之矛
- 这里取意于 EVA 中的命运之矛, 动漫中其可用于攻击卫星轨道上的第15使徒 ARAEL
- 一键日卫星, 是很多安服师傅, 也是我的一个目标之一. 希望我能把信息世界中的这把矛, 做出来
